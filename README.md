# Cipherford - Online Coding Platform

A web-based platform to teach programming in HTML, CSS, JavaScript, Python, and more with real-world projects.

## Features

- Web Development (HTML, CSS)
- JavaScript interactive projects
- LocalStorage for storiing data

## Tech Stack

- HTML, CSS, JavaScript
- LocalStorage
- Git & GitHub

## How to Run

Open `index.html` in your browser.
